# HyChat
- # 介绍

  - HyChat是java nio 基于tcp协议 主从reactor 网络模型 编写的一个高性能单机im 
  - 项目包括服务端和一个简易客户端

- # 功能介绍
    - + [x] 用户登录与鉴权 (基于jwt实现)
    - + [x] 用户在线和下线
    - + [x] 私聊
    - + [x] 群发消息
    - + [ ] 文件传输
    - + [ ] 群聊
